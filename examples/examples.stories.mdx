# Basic Layout Composition

In this page you'll find some examples of basic layout composition.

Minimal styling has been applied to make the examples more readable.

## Product Card

<p id="product-card">
  A simple product card with an image and a description on the the right. Uses
  the [Split](/docs/spacer-components-split--docs), the
  [Stack](/docs/spacer-components-stack--docs), and the
  [PadBox](/docs/wrapper-components-padbox--docs) components.
</p>

The Product card uses the [Split](/docs/spacer-components-split--docs) component to split the image and the description into two columns and the [Stack](/docs/spacer-components-stack--docs) component to stack the description text.

## Pricing Plan

<p id="pricing-plan">
  Examples of pricing plan cards that stack at smaller screen sizes.
</p>

Uses the [Stack](/docs/spacer-components-stack--docs), [Split](/docs/spacer-components-split--docs), [Columns](/docs/spacer-components-columns--docs), and the [PadBox](/docs/wrapper-components-padbox--docs) components.

<Canvas>
  <Story name="Pricing Plan">
    <Columns
      as={PadBox}
      padding="size3"
      gap="size5"
      colCount={3}
      switchAt="sizeMd"
    >
      {plans.map(({ planName, description, price, benefits = [] }) => (
        <PadBox
          key={planName}
          data-plan-card
          as={Split}
          fraction="1/2"
          gap="size7"
          padding={["size9", "size7"]}
        >
          <Stack gap="size5">
            <h2 data-plan-name>{planName}</h2>
            <p data-plan-description>{description}</p>
            <p data-plan-price>{price}</p>
            <button data-plan-button>Buy Plan</button>
          </Stack>
          <Stack data-plan-benefits as="ul" role="list" gap="size2">
            {benefits.map((benefit, i) => (
              <Inline
                key={i}
                data-benefit-item
                as="li"
                align="center"
                gap="size3"
              >
                {benefit}
              </Inline>
            ))}
          </Stack>
        </PadBox>
      ))}
    </Columns>
  </Story>
</Canvas>
